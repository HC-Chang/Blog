Git SSH

發表於 2018-06-25 | 分類於 Resource

Environment

Windows -> Git Bash

Step:

1. Check File

ls -al ~/.ssh

General File Type:

  • id_dsa.pub
  • id_ecdsa.pub
  • id_ed25519.pub
  • id_rsa.pub

2. Make SSH Key

ssh-keygen -t rsa -C “Github Account Mail”

3. Add SSH-Agent

(1) Set SSH-Agent

  • Git Bash

ssh-agent -s

  • msysgit or another

eval ‘ssh-agent -s’

(2) Add SSH Key

ssh-add ~/.ssh/id_rsa

4. Add SSH Key to Github

(1) Copy File

clip < ~/.ssh/id_rsa.pub

(2) Paste File

  • Github Setting
  • SSH and GPG keys

5. Test

ssh -T git@github.com
