Windows Batch

發表於 2017-08-15 | 分類於 Resource

副檔名

.bat / .cmd

常用指令

指令 用途
dir 檔案目錄
del 永久刪除項目
explorer 檔案總管
where 搜尋目標檔案目錄位置
tree 檔案目錄樹狀圖
tree > tree.txt
tree /A \ less (ASCII)
exit 結束
echo 輸出指令
xcopy / robocopy 複製檔案
start 執行完指令後關閉

bat 撰寫

SET 所有環境變數
SET path 單一環境變數

SETLOCAL 設定區域環境變數
ENDLOCAL 結束區域環境變數

SET %MyName% 設定環境變數

P.S. =前後不可有空格

內建環境變數

%CD%
%TIME%
%DATE%

Reference:

  • 保哥-我的 Windows 平台自動化經驗：基礎批次檔撰寫實務
  • 保哥 - Slide

Example:

@ECHO off

ECHO Hello

SETLOCAL
SET mm=apple
ECHO %mm%

SET nn=%mm%-%RANDOM%.txt

ECHO %mm% > %nn%

PAUSE

DEL %nn%

ENDLOCAL
