Environment

Windows -> Git Bash

Step:

1. Check File

ls -al ~/.ssh

General File Type:

  • id_dsa.pub
  • id_ecdsa.pub
  • id_ed25519.pub
  • id_rsa.pub

2. Make SSH Key

ssh-keygen -t rsa -C “Github Account Mail”

3. Add SSH-Agent

(1) Set SSH-Agent

  • Git Bash

ssh-agent -s

  • msysgit or another

eval ‘ssh-agent -s’

(2) Add SSH Key

ssh-add ~/.ssh/id_rsa

4. Add SSH Key to Github

(1) Copy File

clip < ~/.ssh/id_rsa.pub

(2) Paste File

  • Github Setting
  • SSH and GPG keys

5. Test

ssh -T git@github.com

Design Pattern

Veröffentlicht am 2018-05-21 Bearbeitet am 2018-07-26 in Resource

基本原則

SOLID

  • Single Responsibility Principle

    單一功能原則

    • 一個變因

  • Open Close Principle

    開閉原則

    • 擴充開放
    • 修改封閉

  • Liskov Substitution Principle

    里氏代換原則

    • 子類別可替換父類別被使用

  • Interface Segregation Principle

    接口隔離原則

    • 降低類別相依性、耦合性

  • Dependence Inversion Principle

    依賴反轉原則

    • 將物件需求抽象化成介面後再給其他物件繼承使用

  • Demeter Principle

    迪米特原則

    • 物件設計上應減少相互作用，使功能模組相對獨立

  • Composite Reuse Principle

    合成覆用原則

    • 使用合成、減少繼承

  • Simple Factory Pattern - 簡單工廠模式

  • Singleton Pattern - 單例模式
  • Builder Pattern - 建造者模式

  • Iterator Pattern - 迭代器模式

  • Mementor Pattern - 備忘錄模式
  • Observer Pattern - 觀察者模式
  • State Pattern - 狀態模式
  • Null Object Pattern - 空對象模式
  • Strategy Pattern - 策略模式
  • Template Method Pattern - 樣板方法模式
  • Visitor Pattern - 訪問者模式

Windows Batch

Veröffentlicht am 2017-08-15 Bearbeitet am 2018-07-26 in Resource

副檔名

*.bat / *.cmd

常用指令

指令 用途
dir 檔案目錄
del 永久刪除項目
explorer 檔案總管
where 搜尋目標檔案目錄位置
tree 檔案目錄樹狀圖
tree > tree.txt
tree /A | less (ASCII)
exit 結束
echo 輸出指令
xcopy / robocopy 複製檔案
start 執行完指令後關閉

bat 撰寫

SET 所有環境變數
SET path 單一環境變數

SETLOCAL 設定區域環境變數
ENDLOCAL 結束區域環境變數

SET %MyName% 設定環境變數

P.S. =前後不可有空格

內建環境變數

%CD%
%TIME%
%DATE%

Reference:

  • 保哥-我的 Windows 平台自動化經驗：基礎批次檔撰寫實務
  • 保哥 - Slide

Example:

@ECHO off

ECHO Hello

SETLOCAL
SET mm=apple
ECHO %mm%

SET nn=%mm%-%RANDOM%.txt

ECHO %mm% > %nn%

PAUSE

DEL %nn%

ENDLOCAL
